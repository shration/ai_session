새 세션 추가하는 법 — sessions.json만 수정하면 됩니다 index.html 수정 불필요
STEP 1
새 세션 HTML 파일을 GitHub 저장소에 업로드
예: session_03_common.html, session_k1_mkt.html
STEP 2
sessions.json에서 해당 세션의 두 값만 수정
"status": "upcoming" → "status": "done"
"file": "" → "file": "session_03_common.html"
STEP 3
저장 후 GitHub에 커밋 — 끝
index.html은 sessions.json을 자동으로 읽어서 렌더링하므로 별도 수정이 필요 없습니다.
