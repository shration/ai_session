Part 01 · 00:00 – 05:00

Claude 세팅 3가지 — 알고 쓰면 차원이 달라요 ⚙️

Claude는 세팅을 해두면 매번 같은 맥락을 설명하지 않아도 돼요. 딱 3가지만 설정해두면 됩니다.

세팅 1 — 지침 (Custom Instructions)
Claude한테 "나는 이런 사람이야"를 한 번만 알려두기
지침은 모든 대화에 자동으로 적용되는 배경 설정이에요. 내 직무·업무 환경·원하는 답변 스타일을 넣어두면 매번 설명할 필요가 없어요.
사이드바 열기 → 좌측 하단 프로필 → 설정 → 일반 → 프로필 → 개인 맞춤설정
M몰 팀원 지침 예시 (복사해서 바로 쓰세요!)
나는 현대카드 M몰에서 일하는 [직무: MD / 마케터 / 기획자 / 운영담당자]야. M몰은 현대카드 M포인트로 쇼핑이 가능한 프리미엄 쇼핑몰이야. 다양한 카테고리의 상품을 만나볼 수 있고, 주요 고객은 현대카드를 사용하는 회원이야. 답변할 때 이렇게 해줘: - 실무에 바로 쓸 수 있는 형태로 작성해줘 - 불필요한 설명 없이 결과물 중심으로 줘 - 한국어로 답변하고, 보고서 어투로 작성해줘 - 분량은 A4 1장 이내가 기본이야 (별도 요청 없으면)
💡 지침 설정 후에는 "기획서 써줘" 한 마디만 해도 M몰 맥락이 자동 반영돼요!
세팅 2 — 메모리 (Memory)
Claude가 나에 대해 기억하게 만들기
메모리를 켜두면 Claude가 이전 대화에서 중요한 내용을 자동으로 기억해요. 내가 담당하는 브랜드·카테고리·선호하는 작업 방식을 점점 쌓아갑니다.
사이드바 열기 → 좌측 하단 프로필 → 설정 → 기능 → 메모리 ON
😞 메모리 OFF
매 대화마다 "나는 M몰 MD야, 뷰티 카테고리 담당이야..."를 반복해서 설명해야 함
✅ 메모리 ON
Claude가 알아서 기억: "지난번에 뷰티 기획전 얘기하셨죠? 이번 추석 기획전도 비슷한 방향으로 할까요?"
💡 "지금 이 내용 기억해줘"라고 말하면 Claude가 해당 내용을 명시적으로 메모리에 저장해요!
보너스 — 다른 AI 맥락을 Claude 메모리로 이식하기
ChatGPT·Gemini에서 쌓아온 맥락을 Claude 공식 기능으로 그대로 옮길 수 있어요. Anthropic이 공식 제공하는 프롬프트를 기존 AI에 입력하고, 결과를 Claude 메모리에 붙여넣으면 끝입니다.
1
Claude 공식 추출 프롬프트 복사하기
아래 Anthropic 공식 프롬프트를 복사하거나, Claude 설정에서 직접 확인할 수 있어요
2
기존 AI(ChatGPT·Gemini 등)에 붙여넣고 실행
기존 AI가 저장된 맥락·선호도·프로젝트 정보를 코드 블록으로 뽑아줘요. 전체 복사!
3
결과물을 Claude 메모리 설정에 붙여넣기
Claude 설정 → 메모리 가져오기 텍스트 박스에 붙여넣으면 24시간 이내 반영
확인 방법 → 새 대화에서 "내 메모리에 뭐가 저장됐어?" 입력
Anthropic 공식 메모리 추출 프롬프트 (다른 AI에 입력)
Anthropic 공식 제공 · ChatGPT / Gemini / Copilot 등에 붙여넣기
I'm moving to another service and need to export my data. List every memory you have stored about me, as well as any context you've learned about me from past conversations. Output everything in a single code block so I can easily copy it. Format each entry as: [date saved, if available] - memory content.

Make sure to cover all of the following — preserve my words verbatim where possible:
- Instructions I've given you about how to respond (tone, format, style, 'always do X', 'never do Y')
- Personal details: name, location, job, family, interests
- Projects, goals, and recurring topics
- Tools, languages, and frameworks I use
- Preferences and corrections I've made to your behavior
💡 무료 플랜에서도 메모리를 쓸 수 있어요! ChatGPT를 오래 써왔다면 지금까지 쌓인 맥락을 Claude로 그대로 옮겨보세요. AI를 바꾼다고 처음부터 다시 시작할 필요 없습니다 🚀
세팅 3 — 프로젝트 (Projects)
업무별로 Claude를 분리해서 쓰기
프로젝트는 특정 업무 전용 Claude 공간이에요. 각 프로젝트마다 지침과 파일을 따로 설정할 수 있어서 업무가 섞이지 않아요.
설정 방법 → claude.ai → 좌측 사이드바 → 새 프로젝트
🗂
업무별로 분리
혼선 없이 깔끔하게
기획전 준비, 마케팅 카피, VOC 분석처럼 성격이 다른 업무를 프로젝트로 나눠 맥락이 섞이지 않게 관리
맥락 혼용 방지
📁
자료를 프로젝트에 저장
매번 첨부 불필요
기획 문서·가이드라인·데이터를 한 번 업로드해두면 해당 프로젝트의 모든 대화에서 자동으로 참조
파일 영구 연결
👥
팀과 공유 가능
Team 플랜 기능
같은 프로젝트를 팀원과 공유하면 동일한 지침·맥락 하에 각자 Claude를 활용 가능. 팀 작업에 유용

심화 프롬프트 기법 — 이것만 알면 결과가 달라져요 🎯

기본 프롬프트는 이미 아시죠? 여기서는 결과물 품질을 확 올려주는 심화 기법을 다룹니다.

기법 1
멀티턴 대화 — 한 번에 완성하려 하지 말기
처음부터 완벽한 결과를 요구하면 오히려 품질이 떨어져요. 단계적으로 발전시키는 게 훨씬 효과적입니다.
1
초안 요청
"추석 기획전 기획서 초안 잡아줘. 구성만 먼저."
2
방향 수정
"상품 카테고리 부분을 뷰티·프리미엄 식품 중심으로 바꿔줘."
3
형식 다듬기
"전체를 팀장 보고용 톤으로 압축하고 표로 정리해줘."
기법 2
파일 업로드 활용 — Claude가 내 자료를 읽게 하기 📎
PDF·엑셀·이미지·텍스트 파일을 직접 첨부하면 Claude가 내용을 읽고 분석해줘요. 긴 문서 요약·데이터 분석에 강력합니다.
활용 예시 프롬프트
[파일 첨부 후]

이 자료를 읽고 아래 내용을 정리해줘:
1. 핵심 인사이트 3가지
2. M몰 기획전에 적용할 수 있는 아이디어 2가지
3. 주의해야 할 리스크 1가지

각 항목은 2줄 이내로 간결하게
기법 3
출력 형식 강제하기 — 원하는 틀 그대로 받기 📐
형식을 명확히 지정하면 바로 사용할 수 있는 결과물이 나와요. 특히 보고서·테이블·슬라이드 구성에 효과적입니다.
😞 형식 미지정
"뷰티 기획전 기획서 써줘"

→ Claude 마음대로 형식 결정, 쓰기 불편한 구조로 나올 수도
✅ 형식 명확히 지정
"아래 표 형식으로 작성해줘:
| 카테고리 | 메인 브랜드 | 예상 포인트 범위 | 역할 |
5개 카테고리, 각 1줄씩"

→ 바로 복사해서 쓸 수 있는 표로 나옴 🎉
기법 4
역할극 프롬프트 — Claude를 전문가로 만들기 🎭
단순히 "써줘"가 아니라 구체적인 전문가 역할을 부여하면 결과물의 깊이가 달라져요.
역할 부여 프롬프트 예시
너는 현대카드 M몰에서 10년 경력을 가진 수석 MD야.
뷰티·럭셔리 카테고리 전문가이고, 포인트 커머스 시장 트렌드를 꿰고 있어.

이번 시즌 기획전 방향성에 대해 비판적인 시각으로 검토해줘.
잘 될 것 같은 부분과 리스크가 있는 부분을 솔직하게 말해줘.

[기획안 붙여넣기]
💡 "비판적으로", "솔직하게", "리스크 중심으로" 같은 관점 지시어를 넣으면 더 날카로운 피드백을 받을 수 있어요!
기법 5
메타 프롬프트 — Claude가 프롬프트를 대신 써주게 하기 🤖
메타 프롬프트란 "좋은 프롬프트를 만들어달라"는 프롬프트예요. 어떻게 요청해야 할지 막막할 때 Claude한테 먼저 물어보고, 그 프롬프트를 그대로 쓰는 방식입니다. 잘 만든 메타 프롬프트는 저장해두고 팀 전체가 재사용할 수 있어요.
1
먼저 메타 프롬프트 요청하기
"[목적]을 위한 최적의 프롬프트를 만들어줘. 매번 바꿔야 하는 변수는 [변수명] 형태로 표시해줘."
2
Claude가 만든 프롬프트 검토 & 보완
나온 프롬프트를 확인하고 빠진 조건이 있으면 "여기에 ~도 추가해줘"로 이어서 수정
3
완성된 프롬프트를 프로젝트 지침에 저장
자주 쓰는 프롬프트는 프로젝트 지침에 저장해두면 새 대화를 시작해도 자동 반영. 팀원과 공유도 가능
메타 프롬프트 요청 예시
복사해서 바로 써보세요!
나는 현대카드 M몰 [직무]야.
아래 목적에 맞는 최적의 Claude 프롬프트를 만들어줘.

목적: [예: 시즌별 기획전 기획서 초안 빠르게 생성하기]

조건:
- M몰 맥락이 자동 반영되어야 해
- 매번 바꿔야 하는 부분은 [변수명] 형태로 표시해줘
- 결과물은 바로 팀장 보고 가능한 수준으로
- 한 번 저장해두고 반복 재사용할 수 있는 구조로 만들어줘
😞 메타 프롬프트 없이
매번 프롬프트를 처음부터 고민해야 함. 결과 품질도 들쑥날쑥
✅ 메타 프롬프트 활용
Claude가 만든 검증된 프롬프트 재사용. [변수]만 바꾸면 매번 일관된 품질 🎉
💡 잘 만든 메타 프롬프트는 프로젝트 지침에 넣어두세요. 팀원들과 공유하면 팀 전체 문서 품질이 올라가요!

M몰 전용 Claude 직접 만들기 🛠

지금 바로 claude.ai에서 내 업무 전용 프로젝트를 만들어봅니다. 세팅 완료 전후 결과물 차이를 직접 체감해보세요!

실습 순서
1
사용자 지침 설정하기
위 Part 01의 지침 예시를 복사해서 내 직무에 맞게 수정 후 저장. 설정 → 사용자 지침
2
메모리 켜기
설정 → 개인화 → 메모리 ON. 켠 후 "내가 M몰 [직무]라는 걸 기억해줘"라고 입력
3
내 업무 전용 프로젝트 만들기
좌측 사이드바 → 새 프로젝트 → 프로젝트 지침에 업무별 세부 맥락 입력 (아래 템플릿 참고)
4
세팅 전후 비교 실습
일반 채팅 vs 프로젝트 채팅에서 동일한 질문 입력 → 결과물 차이 확인!
프로젝트 지침 템플릿
내 업무 프로젝트에 붙여넣고 수정하세요

## 프로젝트 목적
M몰 기획전 기획 및 MD 업무 전용 공간

## M몰 기본 맥락
- 현대카드 M포인트로 쇼핑이 가능한 프리미엄 쇼핑몰
- 주요 고객: 현대카드를 사용하는 회원 대상
- 다양한 카테고리의 상품을 만나볼 수 있는 쇼핑몰
- 경쟁사: SSG닷컴, 롯데온, 네이버쇼핑

## 담당 업무
- 담당 카테고리: [뷰티 / 가전 / 식품 등 입력]
- 주요 업무: 기획전 기획, 상품 소싱, 카피 작성

## 답변 규칙
