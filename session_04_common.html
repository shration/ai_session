Part 02 · 08:00 – 20:00

핵심 기능 7가지 — 팀에서 이렇게 쓰세요 ⚡

진행자가 Perplexity를 실시간으로 보여주면서 기능별로 M몰 실무 예시를 함께 설명합니다.

기능 01
🎯 소스 설정 — 검색 범위 좁히기
검색창 위 Focus(집중 영역) 클릭 → Web / Academic / Finance / Social / Files+Connectors 선택. 원하는 소스만 검색해 노이즈를 줄여요.
M몰 활용: 경쟁사 뉴스는 Web, 학술 트렌드는 Academic, 금융·주가는 Finance
기능 02
🧑‍💼 개인화 — 내 스타일로 맞추기
좌측 하단 프로필 → Personalize → 직군·답변 스타일·대상 독자를 입력. 이후 모든 검색이 내 스타일로 자동 답변돼요.
팀 표준: "직군: M몰 기획자 / 답변: 3~5줄 요약 + 출처 2~3개 / 독자: 경영진"
기능 03
🤖 모델·모드 선택
검색창 옆 모드 버튼으로 빠른 검색 / 리서치 모드 선택, 모델도 GPT·Claude·Gemini 중 선택 가능. 상황에 맞게 전환해요.
팀 규칙: 회의 전 → 빠른+일반 모델 / 보고서 → 리서치+강한 모델
기능 04
🗂 Spaces — 프로젝트 전용 비서
왼쪽 메뉴 Spaces → Create a Space. 프로젝트별로 모델·지침·파일을 따로 설정하고 팀 링크를 공유해 함께 사용해요.
M몰 예시: "추석 기획전 리서치", "경쟁사 모니터링", "2026 트렌드 분석"
기능 05
⏰ 알림 설정 — 자동 브리핑 받기
Settings → Notifications로 이동해 관심 주제 알림을 설정합니다. 원하는 주제와 수신 방식을 지정해두면 직접 검색하지 않아도 주요 업데이트를 정기적으로 확인할 수 있어요.
실습 과제: "포인트 커머스·경쟁사 동향" 관련 알림을 1개 설정해보기
기능 06
🔍 고급 검색 연산자
따옴표·마이너스·site:·after:/before: 연산자로 검색 결과를 정밀하게 좁혀요. 불필요한 정보를 제거하고 원하는 자료만 빠르게 찾을 수 있어요.
예시: "포인트 쇼핑몰 트렌드" after:2025-01-01 site:news.naver.com -구직
기능 07
🔗 Connectors — 내부 자료 연동
Settings → Connectors → Gmail·Google Drive·Teams·SharePoint·Slack 연동. 외부 웹 검색 + 사내 자료를 한 번에 검색해요.
주의: 연동 전 보안·권한·공유 제한 사항을 팀과 반드시 확인하세요

특히 푸시 알림·앱 내 배너·SNS 연동 측면에서 최신 사례 3가지와 성과 지표를 포함해줘.
M몰 앱 마케팅에 적용 가능한 아이디어도 2가지 제안해줘. 출처 포함.
💻 기획·개발 파트용 — AI 커머스 기술 동향
2025~2026년 커머스 플랫폼의 AI 기술 도입 사례를 알려줘.
특히 AI 추천·개인화·검색 고도화 분야에서 국내외 주요 플랫폼이 적용한 기술과 성과를 정리해줘.
M몰 앱 기획에 참고할 만한 기능 아이디어를 유저스토리 형식으로 2개 작성해줘.

방법 2 — Ollama + 터미널 (고급 사용자용)
Mac·Linux 터미널에서 실행

curl -fsSL https://ollama.com/install.sh | sh

# 2) Gemma 4 E2B 모델 실행 (가장 가벼운 버전)
ollama run gemma4:e2b

# 3) 더 강한 버전 (로컬 GPU 있을 때)
ollama run gemma4:27b
방법 3 — PC에서 Ollama로 다운로드해서 사용하기
1
Ollama 설치

3
PC에서 로컬 AI로 사용
다운로드가 끝나면 인터넷 연결 없이도 PC에서 대화형으로 사용할 수 있습니다. 사양이 낮으면 E2B, 여유가 있으면 E4B 이상을 권장합니다.
PC에서 Ollama 실행 예시

# 노트북·PC 사양이 괜찮을 때

# 설치된 모델 목록 확인
