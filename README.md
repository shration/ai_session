## 📁 파일 구조

```
├── index.html                   ← 전체 커리큘럼 허브 (한 번만 올리면 끝)
├── sessions.json                ← 세션 데이터 (새 세션마다 이것만 수정!)
├── session_01_common.html       ← 세션 01 · 공통 — AI 도구 비교 체험
└── README.md
```

---

## 🚀 새 세션 추가하는 법

1. 새 세션 HTML 파일을 저장소에 업로드
2. `sessions.json` 에서 딱 2줄만 수정

```json
"status": "upcoming"  →  "status": "done"
"file": ""            →  "file": "session_02_common.html"
```

3. 커밋 완료 — `index.html` 은 건드릴 필요 없어요 ✅
