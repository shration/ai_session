NotebookLM 7단계 핵심 기능 라이브 실습 ⚡

진행자가 실시간으로 조작합니다. 7가지 기능을 순서대로 따라하면 2026 트렌드 분석 보고 자료가 완성돼요!

1맞춤 설정 — Customization 2분
AI에게 전문가 페르소나를 부여하기 🎭
단순 요약을 넘어, AI에게 특정 직업과 분석 관점을 부여하면 답변의 깊이가 완전히 달라져요. 채팅 패널의 노트북 구성 아이콘 → 맞춤에서 설정합니다.
1
새 노트북 만들기
notebooklm.google.com 접속 → 새 노트북 클릭 → 제목: "2026 온라인쇼핑 트렌드 분석"
2
소스 추가하기
① 웹 URL — 오픈서베이 리포트, 아임웹 트렌드 가이드 URL 붙여넣기
② 유튜브 링크 — 트렌드 관련 유튜브 영상 URL 붙여넣기
③ 파일 업로드 — PDF 리포트, 사내 기획서 직접 업로드
소스 패널 우측 상단 + 버튼
3
맞춤 설정에 전문가 페르소나 입력
채팅 패널 상단 노트북 구성 아이콘 → 맞춤 클릭 → 아래 프롬프트 입력
맞춤 설정 프롬프트
채팅 패널 → 노트북 구성 아이콘 → 맞춤에 입력
너는 15년차 온라인 마케팅 및 MD 전문가야. 모든 답변은 단순요약을 넘어 비즈니스 기회, 법적규제, 시장침투 전략 관점에서 항상 현대카드 M몰 관점의 시사점을 포함해줘.
오늘 실습에 사용할 소스 URL
NotebookLM 소스 추가 → 웹 URL에 붙여넣기
소스 1) 오픈서베이 온라인쇼핑 트렌드 리포트
https://blog.opensurvey.co.kr/trendreport/online_shopping-2026/

소스 2) 아임웹 2026 커머스 트렌드 가이드
https://imweb.me/blog?idx=439

소스 3) 토스페이먼츠 2026 소비 트렌드
https://www.tosspayments.com/blog/articles/2026trend
💡 맞춤 설정 후 소스를 올리면 이미 전문가 관점이 적용된 노트북 가이드가 자동 생성돼요!
2딥 리서치 — Deep Research 2분
AI가 웹을 직접 탐색해서 자료를 수집해오게 하기 🔍
소스를 수동으로 올리는 것을 넘어, AI가 직접 웹을 탐색해서 관련 자료를 찾아오는 자동화 기능이에요. 수천 페이지 분량의 조사를 몇 분 만에 완료해요.
1
딥 리서치 선택
채팅 패널 왼쪽에서 딥 리서치 선택 → 조사할 주제 입력
2
조사 범위를 구체적으로 지정
뉴스, IT 전문지, 특정 행사(CES, 한국 이커머스 컨퍼런스 등)의 발표 내용을 포함해달라고 구체적으로 요청할수록 품질이 높아져요
딥 리서치 요청 예시
채팅 패널 → 딥 리서치 선택 후 입력
2026년 국내 온라인 쇼핑 시장 트렌드를 조사해줘.
오픈서베이·닐슨·한국인터넷진흥원 등의 최신 리포트, 주요 이커머스 플랫폼 발표 자료, 전문가 분석 기사를 중심으로 수집해줘.
특히 AI 쇼핑·제로클릭·포인트 커머스 관련 내용을 빠짐없이 포함해줘.
💡 딥 리서치 완료 후 수집된 소스가 자동으로 노트북에 추가돼요. 다음 단계에서 이 소스들을 검증합니다!
3소스 검증 — Source Verification 2분
수집된 자료의 신뢰성과 객관성 평가하기 ✅
많이 모은 것보다 좋은 것을 고르는 것이 중요해요. 수집된 소스가 최신 정보인지, 낙관론·비판론 중 어디에 치우쳐 있는지 AI에게 직접 분석시켜봅니다.
소스 검증 질문
채팅 패널에 입력
수집된 소스들이 2025년 이후의 최신 정보인지 확인해주고, 구체적인 수치나 시장 통계 포함 여부도 확인해서 표로 정리해줘.

선별된 소스들이 제조사의 낙관론과 시장전문가의 비판론 중 어디에 더 치우쳐 있는지 분석해서 비교해서 표로 정리해줘.
⚠️ 검증 표에서 2024년 이하 자료, 수치 없는 자료는 다음 단계(필터링)에서 제외할 예정이에요!
4소스 필터링 — Source Filtering 2분
검증된 핵심 소스만 골라서 집중 질문하기 🎯
방대한 자료 중 검증된 소스만 선택해서 질문하면 훨씬 정교한 인사이트를 얻을 수 있어요. 소스 패널에서 체크박스로 직접 선별할 수 있어요.
1
소스 우선순위 정렬
소스 이름 앞에 !를 넣으면 1순위, #을 넣으면 2순위로 상단에 정렬되어 관리가 쉬워져요
2
검증된 소스만 선택
모든 소스 체크박스를 해제 → 3단계에서 검증된 우수 소스만 다시 체크 → 해당 소스만 선택된 상태에서 질문 입력
필터링 후 집중 분석 질문
핵심 소스만 선택한 상태에서 채팅 패널에 입력
선택된 소스만을 바탕으로 2026 온라인 쇼핑 트렌드 핵심 5가지를 정리해줘.
각 트렌드마다: ① 트렌드명 ② 핵심 내용 1~2줄 ③ 관련 수치나 사례 ④ M몰 관점에서의 비즈니스 기회와 법적 리스크를 포함해줘.
💡 전체 소스보다 검증된 소스 3~5개에 집중했을 때 더 날카롭고 실용적인 인사이트가 나와요!
5멀티모달 전략 — Multi-modal Strategy 2분
PDF + 유튜브를 합치면 새로운 시야가 생겨요 🎬
텍스트(PDF, 웹)뿐만 아니라 유튜브 영상을 소스로 추가하면 완전히 다른 관점이 더해져요. PDF는 학술적 분석, 유튜브는 현장의 생생한 목소리를 담고 있어요.
1
유튜브 소스 추가
소스 패널 → + 버튼 → 유튜브 URL 붙여넣기
트렌드 관련 유튜브 영상, 전문가 인터뷰, 커머스 컨퍼런스 발표 영상을 추가해보세요
유튜브 자막이 있는 영상이어야 분석 가능
2
PDF 소스 + 유튜브 소스 함께 선택 후 질문
두 유형의 소스를 모두 체크한 상태에서 아래 프롬프트를 입력하면 관점이 교차된 깊은 분석이 나와요
멀티모달 통합 분석 질문
PDF 소스 + 유튜브 소스 함께 선택 후 채팅 패널에 입력
리서치 보고서의 데이터·통계와 유튜브 리뷰에서 나타나는 소비자 Pain Point를 결합해서
M몰이 2026년에 실행할 수 있는 새로운 서비스 전략 3가지를 제안해줘.
각 전략마다 시장침투 가능성과 예상 리스크도 함께 분석해줘.
💡 데이터(PDF) × 현장 목소리(유튜브) 조합이 가장 설득력 있는 보고 자료를 만들어줘요!
6AI 오디오 오버뷰 — Audio Overview 2분
AI 두 명이 M몰의 리스크와 기회를 치열하게 토론해줘요 🎙️
분석된 내용을 두 사람의 팟캐스트 스타일 대화로 만들어요. 보고 전 최종 검토, 출퇴근 중 청취, 팀원 공유 등 다양하게 활용할 수 있어요.
1
스튜디오 패널 열기
오른쪽 스튜디오 패널 → 오디오 오버뷰 선택
2
> 아이콘 클릭 → 맞춤 설정에 입력
오디오 오버뷰 옆 > 아이콘을 클릭 → 아래 프롬프트 입력 후 생성. 2~3분 대기
오디오 오버뷰 맞춤 프롬프트
스튜디오 → 오디오 오버뷰 → > 아이콘 → 맞춤 설정에 입력
한 명은 깐깐한 임원, 한 명은 열정적인 팀장 역할을 맡아
2026 온라인 쇼핑 트렌드가 M몰에 주는 리스크와 기회에 대해 치열하게 토론해줘.
특히 AI 쇼핑과 제로클릭 쇼핑 트렌드에서 M몰이 준비해야 할 것과 놓칠 수 있는 위험 요소를 중심으로 논쟁해줘.
💡 완성된 오디오는 공유 링크로 팀원에게 바로 전달하거나, 출퇴근 시간에 귀로 들으며 복습할 수 있어요!
7시각화 자료 제작 — Visualization 3분
분석 결과를 슬라이드·인포그래픽으로 자동 완성 🎨
6단계까지 쌓인 분석 내용을 바탕으로 보고용 슬라이드(PPT)나 인포그래픽을 자동 생성해요. 글자가 깨지거나 수정이 필요하면 Gemini Canvas로 가져가 편집할 수 있어요.
1
슬라이드 자료 생성
스튜디오 패널 → 슬라이드 자료 선택 → 아래 프롬프트 입력 → 2~3분 대기 후 PPTX 다운로드
2
인포그래픽 생성 (보너스)
스튜디오 패널 → 인포그래픽 선택 → 핵심 수치·일정·체크리스트를 대시보드 형태로 요약 요청
3
Gemini Canvas로 후편집
슬라이드 텍스트가 깨지거나 수정이 필요하면 → 텍스트 복사 → Gemini Canvas로 가져가 수정 가능한 PPT로 변환
gemini.google.com → Canvas 모드
슬라이드 생성 커스텀 프롬프트
스튜디오 → 슬라이드 자료 → > 아이콘에 입력
